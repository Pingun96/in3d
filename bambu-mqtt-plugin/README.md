# bambu-mqtt-plugin

MQTT Plugin

## Install

To use npm

```bash
npm install bambu-mqtt-plugin
````

To use yarn

```bash
yarn add bambu-mqtt-plugin
```

Sync native files

```bash
npx cap sync
```
